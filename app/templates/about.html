About

This OAI Data Provider adheres to the OAI standard and supports all the verbs (Identify, ListMetadataFormats, ListSets, ListIdentifiers, ListRecords, and GetRecord), resumption tokens, and sets. Only DC metadata is provided. Sets correspond to the archival collections in the Caltech Archives.

Data is delivered from a static XML repository that is updated from ArchivesSpace daily, at 1am Pacific time. Collections and records without attached digital content are not included.

Source code is available on GitHub

For more information visit:

  • OAI-PMH specifications
  • Caltech Archives website
