About

This OAI Data Provider adheres to the OAI standard and supports all the verbs (Identify, ListMetadataFormats, ListSets, ListIdentifiers, ListRecords, and GetRecord), resumption tokens, and sets. Only DC metadata is provided. Sets correspond to the archival collections in the Caltech Archives.

Data is delivered from a static XML repository that is updated from ArchivesSpace daily. The static repository is derived from ArchivesSpace Encoded Archival Description (EAD) output for all collections that contain links to digital content. Collections and records without attached digital content are not included.
