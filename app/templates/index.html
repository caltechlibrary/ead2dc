OAI-PMH Data Provider

Base URL: https://apps.library.caltech.edu/ead2dc/oai

This data provider supports all OAI-PMH verbs, and most arguments, viz.:

  • Identify
    • example: Identify
  • ListMetadataFormats
    • argument not supported: identifier
    • example: ListMetadataFormats
  • ListSets
    • argument not supported: resumptionToken
    • example: ListSets
  • ListIdentifiers
    • arguments supported: set, from, until, resumptionToken
    • argument not supported: metadataPrefix
    • example: ListIdentifiers&set=HaleGE&from=2023-01-01&until=2023-10-31
    • example: ListIdentifiers&resumptionToken=HaleGE%7C2023-01-01%7C2023-10-31%7C250
  • ListRecords
    • arguments supported: set, from, until, resumptionToken
    • argument not supported: metadataPrefix
    • example: ListRecords&set=Photographs
    • example: ListRecords&resumptionToken=Photographs%7C000-00-00%7C999-99-99%7C250
  • GetRecord
    • argument supported: identifier
    • argument not supported: metadataPrefix
    • example: GetRecord&identifier=collections.archives.caltech.edu/repositories/2/archival_objects/106628

Notes:

  • Metadata
    • Dublin Core is the only metadata standard supported at this time.
  • Flow control
    • ListIdentifiers and ListRecords return batches of 250 records, with a resumption token.
    • Resumption tokens are valid indefinitely. Note, however, that idempotency is not guaranteed as the underlying data may change at any time.
  • Sets
    • The set structure is flat, not hierarchical.
    • All records belong to one and only one set, so harvesting all sets will capture all records.
