Introduction

This Data Provider allows external services to harvest metadata for digital collections hosted by Caltech Archives. The DP adheres to the Open Archives Initiative Protocol for Metadata Harvesting (OAI-PMH) and supports all the verbs (Identify, ListMetadataFormats, ListSets, ListIdentifiers, ListRecords, and GetRecord), resumption tokens, and sets.

Only Dublin Core metadata is available at the current time.

Sets correspond to the archival collections in the Caltech Archives.

Data is delivered from a static XML repository that is updated from ArchivesSpace daily, at 1am Pacific time. Only collections and records with digital content accessible on the web are included.

Access to the configuration section requires an Archives staff login.

DP Base URL

Harvesting queries are formed by adding OAI-PMH parameters to this base URL:

  • https://apps.library.caltech.edu/ead2dc/oai

The DP supports the six OAI verbs and their arguments, with a few exceptions, as described below. The service is freely available. Any questions about this service should be directed to archives@caltech.edu

Example queries:

  • Identify
    • Identify
  • ListMetadataFormats
    • ListMetadataFormats
    • ListMetadataFormats&identifier=collections.archives.caltech.edu/repositories/2/archival_objects/106578
  • ListSets
    • ListSets
      • argument not supported: resumptionToken
  • ListIdentifiers
    • ListIdentifiers
    • ListIdentifiers&set=resource_30
    • ListIdentifiers&set=resource_30&from=2025-01-01
      • arguments supported: set, from, until, resumptionToken
      • argument not supported: metadataPrefix
  • ListRecords
    • ListRecords
    • ListRecords&set=resource_219
    • ListRecords&set=resource_30&from=2025-01-01
      • arguments supported: set, from, until, resumptionToken
      • argument not supported: metadataPrefix
  • GetRecord
    • GetRecord&identifier=collections.archives.caltech.edu/repositories/2/archival_objects/106628
      • argument supported: identifier
      • argument not supported: metadataPrefix

Additional notes

  • Metadata
    • Dublin Core is the only metadata standard supported at this time. Additional metadata formats may be added in the future as the need arises. Comments, advice and corrections about the metadata are welcome.
  • Flow control
    • ListIdentifiers and ListRecords return batches of 250 records, with a resumption token.
    • Resumption tokens are valid indefinitely. Note, however, that idempotency is not guaranteed as the underlying data may change at any time.
  • Sets
    • Sets correspond to Caltech Archives collections. No other set specifications are supported at this time.
    • The set structure is flat, not hierarchical. i.e. There are no sets representing portions of collections.
    • All records belong to at least one set, so harvesting all sets will capture all records.
    • Records may belong to more than one set (i.e. collection).
